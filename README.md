# Fast API Crud Application

Fast API implementation with relational database mysql.

#### Setup project

Git clone repo from git

```bash 
git clone https://github.com/navjotgwal/fast_api_rel_db.git
```

Install dependency 

``` bash
basgpip install -r requirements.txt
```
